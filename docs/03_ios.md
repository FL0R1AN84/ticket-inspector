# iOS Development

## General infos for local development

For the first build (includes livereload) on local simulator devices

```bash
npm run build
```

### Requirements

- Ruby

```bash
brew install ruby
```

- CocoaPods

```bash
brew install cocoapods
```

```bash
npm install @capacitor/ios
```

```bash
npx cap add ios
```

### Device simulator

```bash
npx cap run ios
```
